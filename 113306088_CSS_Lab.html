我的學習旅程：第一次製作網頁 在學習網頁設計的這段時間裡，我投入了許多心血和努力。為了完成這個作業，我不僅學會了 HTML 的基本語法，還深入研究了 CSS 的排版與樣式設計。雖然一開始面對複雜的語法和層層嵌套的結構時感到手足無措，但我並沒有因此退縮。 我首先仔細觀察作業的要求，將頁面設計分解為多個小目標，逐一攻克。在寫導航列時，我花了很長時間研究如何使用 ul 和 li 實現清晰的結構，同時還要考慮到下拉選單的互動效果。這對我來說是一個挑戰，因為它需要將 HTML 和 CSS 無縫結合。 當遇到下拉選單無法正常顯示或樣式跑版時，我一次次調整程式碼，甚至參考了不少教學資源和範例。每當解決一個問題時，我都感受到滿滿的成就感，也逐漸對網頁設計產生了興趣。 這次作業不僅是完成課堂任務，更是一段自我成長的旅程。透過不斷的嘗試與錯誤，我學會了如何用程式語言實現心中的想法。雖然這只是我邁向網頁設計世界的第一步，但我為自己的努力感到自豪，也期待在未來能夠創造出更多精彩的作品。
